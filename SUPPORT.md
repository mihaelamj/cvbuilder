# Support

Need help with CVBuilder? Here is where to go.

- **Bug reports**: open an [issue](https://github.com/mihaelamj/cvbuilder/issues/new/choose)
  using the bug form.
- **Feature requests**: open an issue using the feature form.
- **Security issues**: do not open a public issue. See [SECURITY.md](SECURITY.md).

Before filing, please search existing issues to avoid duplicates.

CVBuilder is pre-1.0 and maintained on a best-effort basis. Clear, reproducible
reports get the fastest responses.
