# CVBuilder v0.1.0 Release Notes

Status: draft for the first public release tag.

These notes are prepared for `v0.1.0`. Publish them after the release commit is
on `main`, the `v*` tag checks pass, and the maintainer creates the GitHub
Release.

## What Ships

CVBuilder `v0.1.0` is a pure Swift, Markdown-first technical CV generator for
macOS and Linux.

Included behavior:

- `CVDocument` as the canonical JSON data contract for publishable CV data.
- Deterministic Markdown rendering from structured Swift or JSON data.
- `cvbuilder` CLI for Markdown output, normalized JSON output, and `--check`.
- Linux-only `CVBuilderTileDown` adapter for TileDown-compatible Markdown
  workflows.
- JSON Schema metadata at `Schemas/cvdocument.schema.json`.
- Realistic example and fixture coverage for technical CV variants.
- Rendering modes for experienced, early-career, and public-evidence-heavy
  technical CVs.
- Checked-in release hygiene: style, namespacing, platform contract, schema
  drift, fixture freshness, consumer smoke, macOS Swift, and Linux Swift gates.

## Build

```sh
swift build
swift build --product cvbuilder
```

Run the CLI:

```sh
swift run cvbuilder --data cv.json --out cv/index.md
swift run cvbuilder --data cv.json --out cv.normalized.json --format json
swift run cvbuilder --data cv.json --out cv/index.md --check
```

## Release Gates

The release commit must pass:

- Style and namespacing.
- Platform contract check.
- Swift macOS workflow.
- Swift Linux workflow.
- Schema drift check.
- Generated fixture freshness check.
- Clean SwiftPM consumer smoke test.

On Linux, the consumer smoke test also imports `CVBuilderTileDown` and compares
its output with canonical Markdown.

## Product Boundary

This release is Markdown and JSON only.

CVBuilder core does not include:

- PDF generation or rendering.
- HTML rendering.
- static-site generation or publishing.
- ATS scoring, resume optimizer claims, or parser gaming hints.
- photos, demographic metadata, personality labels, culture-fit labels, fit
  scores, skill bars, tables, or column layout in the canonical renderer.
